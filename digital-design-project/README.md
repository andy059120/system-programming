# digital-design-project

### 微波爐時間與火力設計專案

```
A：左上角紅黃綠 LED 12顆:顯示微波爐火力強度(選擇模式 大中小火)
當機器開啟或關閉時，可以做出好看的效果(使用ripple counter)

B：共陰極七段顯示器:有六個數字的顯示

左往右數兩個數字:目前模式的設定
後面四個數字:時間的倒數，以秒鐘表示(序向邏輯的實作)

C：蜂鳴器(BUZZER) :在七段顯示器右邊
蜂鳴器使用的時間點
1.倒數計時歸0時
2.開機關機時
3.按火力強度時就叫一下

F：時脈電路clock:輸出系統時脈OSC(時脈的VCC)

H：8X8 點矩陣 LED 顯示器(主機板右邊)
ex:當你要加熱三明治時，顯示一個三明治的形狀
ex:當你要加熱雞腿時，顯示一個雞腿的形狀

J：資料開關(DATA SWITCHIES)*8(左下角):

SW1:視為整體的電源開關
SW2:開始加熱的按鈕
SW8:緊急中斷按鈕(所有的時間都暫停)


控制七段顯示器的功能

K：脈波按鍵4顆(PULSE):控制七段顯示器的功能
PS1:數字+1
PS3:數字-1
PS2:選擇七段顯示器(往前)
PS4:選擇七段顯示器(往後)
當下選擇的七段顯示器亮右下角的燈
```
